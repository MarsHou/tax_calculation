# Tax Calculation

根据2018年10实行的个税新计算方式开发的一个手机app，支持iOS、Android端。
使用的是Flutter开发。enjoy！

## Getting Started

1、先装好flutter环境，安装步骤请参考[Flutter官网](https://flutter.dev/docs/get-started/install)

2、clone 本项目, 电脑链接手机。
```shell
git clone git@github.com:MarsHou/tax_calculation.git
cd tax_calculation
flutter run
```
